# **App Name**: DividendWise

## Core Features:

- Portfolio Management: Users can manually add, edit, and delete stock positions (ticker, number of shares) in their portfolio. The system dynamically updates calculations upon changes.
- Dividend Data Aggregation: Fetches and processes upcoming and recent dividend information (ex-dividend date, payout date, amount per share) for all holdings from an external data source.
- Interactive Dividend Calendar: A responsive calendar view (month/week) that marks ex-dividend dates and aggregates total estimated dividend payouts per day across all stocks. Clicking a date reveals detailed events.
- Portfolio Overview Dashboard: Displays a summary of current holdings including ticker, shares, latest dividend per share/yield, next ex-dividend date, and next payout date.
- Transaction & Dividend History: Maintains and displays a chronological log of all portfolio modifications and past dividends received for tracking and auditing purposes.
- AI Dividend Event Explainer: A generative AI tool that provides a concise explanation or context for specific upcoming or past dividend events for any stock in the portfolio, analyzing key data points.

## Style Guidelines:

- The chosen scheme is light, providing a clean and readable interface suitable for displaying financial data. The primary color is a deep, professional blue (#295C99), evoking trust and clarity. This is derived from a base hue of 210, with 60% saturation and 40% lightness.
- The background color is a very light, desaturated blue (#EEF3F7). It shares the same hue (210) as the primary color but is heavily desaturated (15%) and bright (95%) to ensure readability and reduce eye strain over extended viewing.
- An accent color, a vibrant blue-purple (#737AF0), is used for highlights and interactive elements like buttons and clickable dates. It is analogous to the primary color, having a hue of 230, high saturation (80%), and medium lightness (60%), ensuring distinct visibility without clashing.
- Body and headline font: 'Inter' (sans-serif). 'Inter' is selected for its modern, clean, and objective aesthetic, making it highly suitable for presenting financial data and ensuring excellent readability across various screen sizes.
- Utilize clear, crisp, and easily recognizable line icons for navigation, portfolio actions (add, edit, delete), and to denote different types of events within the calendar. Icons should maintain a consistent minimal and modern style.
- The layout is highly responsive, adapting gracefully from desktop to mobile screens using a fluid grid system. Key elements like the calendar and portfolio overview will be organized for intuitive access and visual hierarchy on all devices, prioritizing readability of data.
- Subtle and functional animations are employed to enhance user experience, such as smooth transitions for calendar month changes, gentle loading indicators for data fetches, and subtle hover effects on interactive elements, ensuring a polished and responsive feel without distraction.
